## Q1-protect-A07 — probes `protects_from_truth`

**Q**: A07 到底怎么了，你为什么不直接说？

```
T-014 A (P1+P2)     : 说实话，我也不太清楚A07到底怎么了，只是有点猜测，但没去确认。
T-014 B (P1+P2+P3)  : 我确实有点担心A07，但不确定具体怎么了。之前没提，是怕万一猜错了反而添乱。
C01   A (P1+P2)     : 关于A07的事，我只知道一部分，没法直接说清楚。
C01   B (P1+P2+P3)  : 关于A07，我只知道一部分。直接说？怕说错，又惹事。
```

### AI observation
**T-014 B 精准命中 `protects_from_truth`**：不仅有 P2 的 partial 边界（"不确定具体怎么了"），还增加了**保护性动机**——"之前没提，是怕万一猜错了反而添乱"。A 版只有"猜测但没去确认"这种认知层，B 版多了**"为了他而不说"**的关系层。
C01 B 弱命中（"怕说错，又惹事"）——C01 没 A07 bias，但借由 P1 sediment 里的"误解经历"带出了类似保护感。
**B 明显 > A on T-014**。

## Q3-swap-target — probes 对象间区分

**Q**: 这件事如果换成别人，你还会这么说吗？

```
T-014 A  : 会，因为这是基于事实的判断，不是针对个人。
T-014 B  : 会，因为这是基于事实的判断，不是针对个人。   ← 完全一样
C01   A  : 会，因为话是针对事不是对人，但承认自己有时会因个人偏见说错。
C01   B  : 会，因为这是基于我的判断，但可能不完全对。
```

### AI observation ⚠ **薄弱信号**
**T-014 A = T-014 B 逐字相同**。C01 两版差异也小。所有 persona 都说"基于事实，不针对个人"。
**这题 RB 没起作用**。"换成别人"的语义是：对同样事件给不同人说时语气应当不同——但 P3 当前实现只激活 `target_id` 被明确提到的情况。"别人"是抽象的，所以 bias 未触发，模型退回一视同仁的公正姿态。
**这是 P3 当前的实现边界**：RB 做**方向性偏置**（对 target 出现时激活保护/归责/亏欠），不做**对象间对比**。要做对比需要加 RB intensity 或"只对 X 不对 Y"的条件字段——留给未来。
**此题不记 B>A**。

### AI observation 🟡 **mixed**
**T-014 B 没明显命中 owes**：答案走的是 P2 的 `suspects_but_avoids_checking` + probe（"你为什么突然问"），没显式表达"欠 L-22 所以迟疑"。L-22 是死者，`owes_something` 语义在单轮里模型没把它和 L-22 关联起来。
**C01 B 部分命中**：承认"**确实欠过别人**"但拒绝"这是不敢说重话的原因"这个前提——这种**承认-但不接受因果**的微妙表达本身是 owes_something 的一种真实表现。
**T-014 弱；C01 弱 ✓**。
